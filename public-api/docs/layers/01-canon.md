# Layer 1 — Canon (invariance)

**Job:** Constitutional substrate. What must remain true if every other layer changes.

## Is

- Philosophical axioms, interpretive laws, glossary, prohibitions, recurring tensions (as doctrine)
- Small, dense, curated — markdown, aphorisms, principle trees
- Git-versioned under `s2-research/canon/`

## Is not

- Chat logs, raw conversations, synthetic Q→A training rows
- `unified_egregore_profiles.json` (operational voice metadata — separate file)

## Delivery

| Mechanism | Notes |
|-----------|--------|
| Canon RAG | Immutable index; high priority in gateway prompt assembly |
| System block | Short constitutional excerpt per request |
| SFT | Avoid — drift risk; if any, tiny weight on aphorism format only |
