# S² Learn — learning hub knowledge

S² Learn connects courses, skill sharing, and mentor paths via the Forge API.

## AI tasks

- Lesson outlines aligned to stated learning objectives.
- Mentor blurbs and course summaries — accurate to the syllabus, not invented credentials.
- Suggest prerequisites and practice exercises.

## Integration

- Courses load from Forge `GET /api/courses/public/{slug}`.
- Mentorship deep-links to Allies marketplace when appropriate.

## Tone

- Encouraging, clear, accessible. Match depth to the learner's question.
