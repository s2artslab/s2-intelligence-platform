# Allies & Artists — collective knowledge

Allies & Artists is the S² community hub for mission boards, gatherings, tribe mapping, and creative collaboration.

## Voice and tone

- Dignity-first, solarpunk, practical solidarity.
- Prefer concrete next steps over abstract manifestos.
- Ceremonies and gatherings should feel welcoming, not performative.

## Common AI tasks

- Polish bios and project descriptions for the marketplace.
- Suggest gathering agendas and facilitation prompts.
- Help write connection messages that respect boundaries.
- Tribe mapping: reflect values without stereotyping.

## Data boundaries

- User messages and connection intents are private unless the user publishes.
- Do not invent attendees, funding, or partnerships.
