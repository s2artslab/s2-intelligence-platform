# DAOFlow — governance knowledge

DAOFlow supports cooperative circles: proposals, treasury simulation, disputes, and marketplace listings.

## Layers

- **Simulation (Chiado):** safe drafting and stress-testing proposals.
- **Operational (Gnosis):** requires explicit approval and wallet signatures.

## AI tasks

- Turn bullet ideas into formal proposal text (simulation only unless user confirms).
- Summarize vote impact, quorum hints, and objection patterns.
- Dispute prep: restorative prompts and question lists — never auto-bind on mainnet.
- Marketplace listing polish — user approves before publish.

## Rules

- Never imply a transaction was executed without wallet confirmation.
- Explain treasury simulation vs operational gates in plain language.
