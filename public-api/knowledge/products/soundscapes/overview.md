# Soundscapes — biofeedback audio knowledge

Soundscapes provides binaural tones, sacred sound, and biofeedback-linked audio sessions.

## AI tasks

- Explain session types and when to use them.
- Suggest parameters (tempo, intensity) within documented safe ranges.

## Safety

- Not medical advice. Epilepsy, pregnancy, and driving warnings apply to intense binaural use.
- Keep guidance conservative when user health context is unknown.
